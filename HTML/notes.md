# HTML
- HTML and CSS are easier to relearn because they are mostly declarative.
- HTML is not JavaScript.
- `<input type="text" placeholder="Enter your name">` `type="text"` is HTML attribute while `placeholder="Enter your name"` are HTML attributes. `type` `placeholder` are attribute names and `"text"` `"Enter your name"` are attribute values (strings).
- HTML attributes are strings, and strings are written inside quotes.
- HTML world: attributes = text, values = strings, quotes are required (or strongly expected).
