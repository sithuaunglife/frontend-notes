# DigitalOcean

## Facts
- Next.js can run on RAM 1 GB / 1 CPU server. 8 GB / 4 CPU is generally recommended.
- When you create a server using a DigitalOcean droplet, it gives your server a public IP address. If you put your built website files inside the server and configure a web server like Nginx, the website can be accessed using that IP address.
