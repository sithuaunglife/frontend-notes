# Node.js

## Facts
- Node.js is a JavaScript runtime that allows JavaScript to run outside the browser.
- Node.js is commonly used for backend development, APIs, servers, and command-line tools.
- Node.js can also be used for Internet of Things (IoT) applications, but it is not limited to IoT.
- Node.js allows you to run JavaScript outside the browser.
- You can use Node.js in the terminal to execute JavaScript files on your computer.
- Node.js is commonly used for backend development, scripting, automation, and development tools.
- Without Node.js (or another JavaScript runtime), you cannot run JavaScript directly from the terminal.
- The terminal itself doesn't understand JavaScript. It needs a runtime to execute it.
- Package managers like npm and pnpm use Node.js behind the scenes to run JavaScript tools.
- Commands like `npm run dev` and `pnpm dev` start Node.js automatically, so you rarely need to run `node` command manually.
- Node.js comes with npm (Node Package Manager) installed by default.
- npm is used to install and manage JavaScript packages.
