# Misc notes

## Facts
- There are no-code tools that only require design skills. You don't need to code. You can prompt AI to generate code with Wix, Webflow, and Framer.
- Vue.js is a JavaScript framework for building user interfaces.
- Vue.js is popular, but React has a larger ecosystem and job market globally.
- Nuxt.js is a framework built on top of Vue.js for server-side rendering (SSR), static sites (SSG), and full-stack web applications.
- Nuxt.js is less popular than Next.js because React has a larger market share.
- Angular is a frontend framework developed by Google.
- Angular is commonly used in large enterprise applications.
- Angular has a smaller community and job market than React globally.
- Angular is less commonly chosen for new projects than React in many regions, but it remains actively maintained by Google and is still widely used in enterprises.
- If you know React, learning React Native is easier because they share many concepts.
- React Native is a cross-platform framework for building iOS and Android applications from a single codebase.
- React Native is suitable for many production applications.
- For apps requiring the highest performance or deep platform-specific features, native development (Swift for iOS and Kotlin/Java for Android) may be a better choice.
- Electron is a framework for building cross-platform desktop applications.
- Electron allows developers to build desktop apps using HTML, CSS, and JavaScript.
- Electron uses web technologies and runs on Windows, macOS, and Linux from a single codebase.
- Phaser is a JavaScript framework for building 2D browser games.
- Three.js is a JavaScript library for rendering 3D graphics on the web.
- Babylon.js is a JavaScript 3D engine for building games and interactive 3D applications.
- Phaser, Three.js and Babylon.js are commonly used for browser-based games and interactive web experiences.
- Large AAA games are typically built with engines like Unity or Unreal Engine rather than JavaScript.
- Vibe coding is a development style where developers use AI to generate, modify, and iterate on code through natural language prompts.
- Developers are still responsible for understanding, reviewing, testing, and maintaining the generated code.
- Vibe coding can speed up development, but it should not replace understanding the code.
- RAM (Random Access Memory) is temporary memory that stores data and programs while the computer is running.
- HDD (Hard Disk Drive) and SSD (Solid State Drive) are permanent storage devices that keep data even after the computer is turned off.
- The CPU (Central Processing Unit) executes instructions, performs calculations, and coordinates the computer's operations.
