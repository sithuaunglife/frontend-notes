# SWR

## Facts
- `shouldRetryOnError` (default: true): Automatically retries failed requests. Set to `false` to prevent retries (e.g. server is down).
- `revalidateOnFocus` (default: true): Automatically refetches data when the browser window/tab regains focus. Set to `false` to disable this behavior.
